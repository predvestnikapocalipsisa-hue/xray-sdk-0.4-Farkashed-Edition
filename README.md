### X-Ray SDK 0.4 Farkashed Edition

Модифицированная версия X-Ray SDK 0.4, которая расширяет возможности, делает инструментарий более удобным и исправляет баги. Проект базируется на TSMP SDK (который в свою очередь основан на X-Ray SDK 0.8 by Red Panda).

### Основные преимущества:

    Архитектура x64: Полная поддержка 64-битных систем, что значительно ускоряет работу и позволяет работать с тяжелыми сценами.

    Современный UI: Интерфейс на базе ImGui с кастомизацией цветов и набором тем.

    Автономность: Не требует установки Visual C++ для запуска и собирается в Visual Studio 2019 без лишних зависимостей.

    Из репозитория вырезано всё, кроме самих редакторов и необходимых компонентов.

### Список изменений:

    Исправление ошибок TSMP SDK и SDK 0.8 для достижения полной работоспособности.

    Исправлены вылеты и баги при создании растительности на локации

    Поддержка x64 архитектуры.

    Значительно более удобный интерфейс с кастомизацией тем.

    Исправлен краш при закрытии Level Editor.

    Исправлен краш при нажатии Apply/OK в текстовых свойствах.

    Исправлен краш при редактировании текстовых свойств нескольких объектов одновременно.

    Добавлены уникальные иконки для редакторов.

    Реализовано создание превью-картинок и генерация LOD-текстур в Library Editor.

    Реализована поддержка ресурсов ТЧ для запуска Particle Editor.

    Исправлено окно Minimap Editor (теперь закрывается корректно).

    Исправлено несохранение настроек окон при наличии кириллицы в путях.

    Исправлены неверные координаты спавна при качестве рендера отличном от 100%.

    Добавлен расчет номера билда по аналогии с игровым движком.

    Исправлен вылет при выделении порталов.

    Исправлена генерация порталов 

    Обновлен ImGui и переделано окно свойств (все ветки раскрыты по умолчанию) 

    Переделано окно выбора элементов: теперь используются чекбоксы 

    Лимит команд в rpoint расширен до 31 

    Исправлены проблемы с путями, если в названии папки есть точка 

    Починена возможность множественного выбора в диалоге файлов 

    Исправлено неправильное поведение окна подтверждения в синхронизации текстур/звуков 

    Реализована основа Library Editor 

    Вырезан Boost и обновлен BugTrap для повышения стабильности 

    Исправлен сломанный стек вызовов в дампе и при отладке 

    Починена работа детальных объектов (трава, листья) и их генерация 

    Исправлено предложение сохранить сцену, если в ней ничего не менялось 
    
    Адаптация для совместимости с оригинальными компиляторами и движком ТЧ 

    Исправлена ошибка выбора файлов при наличии папки и файла с одинаковым именем 

    Исправлен краш при переименовании waypoint или их рендере в одной точке 

    Починено отображение выделения партиклов 

### Установка

Ставить поверх оригинального XRAY SDK 0.4 с заменой   

### Сборка

Проект настроен на сборку в Visual Studio 2019. Просто откройте .sln файл и выберите конфигурацию Release | x64.
