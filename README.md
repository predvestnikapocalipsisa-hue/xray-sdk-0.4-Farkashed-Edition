### X-Ray SDK 0.4 Farkashed Edition

Модифицированная версия X-Ray SDK 0.4, которая расширяет возможности, делает инструментарий более удобным и исправляет баги. Проект базируется на TSMP SDK (который в свою очередь основан на X-Ray SDK 0.8 by Red Panda).

### Основные преимущества:

    Архитектура x64: Полная поддержка 64-битных систем, что значительно ускоряет работу и позволяет работать с тяжелыми сценами.

    Современный UI: Интерфейс на базе ImGui с кастомизацией цветов и набором тем.

    Автономность: Не требует установки Visual C++ для запуска и собирается в Visual Studio 2019 без лишних зависимостей.

    Расширенный функционал: Новые фичи а так же исправление оригинальных недоработок.

    Стабильность: В отличии от оригинального сдк, модифицированная версия намного стабильнее и вызывает меньше проблем при использовании.

    Из репозитория вырезано всё, кроме самих редакторов и необходимых компонентов.

### Список изменений:

    Исправление ошибок TSMP SDK и SDK 0.8 для достижения полной работоспособности.

    Исправлены вылеты связанные с окном chooseform.

    Добавлен выбор спектра обьектов в обджект лист через shift + лкм.
    
    Исправлен вылет при попытке переместить предмет.
    
    Исправлен неудобный баг, из-за которого object list обрезался.

    Изменены некоторые иконки на топ баре [ВЗЯТО ИЗ OGSR & B.O.R.S.C.H.T SDK].

    Исправлены проблемы с секторами.
    
    Исправлены вылеты связанные со скриптами.
    
    Исправлены проблемы с торговцами.

    Исправлены проблемы с шейдерами из-за которых баговался рендер. (подробнее в вики).

    Добавлен тхм генератор для террейнов, благодаря которому были исправлены проблемы с .thm файлами из-за которых баговался рендер. (подробнее в вики).

    Исправлен баг из-за которого сдк не видел буфер обмена системы.

    Интерфейс стал красивее и удобнее, добавлена кастомизация и темы.

    Починена работа детальных объектов (трава, листья) и их генерация.

    Улучшено качество рендера.

    Добавлены вкладки со сценами как в браузере.
    
    Новые иконки редакторов.
    
    Добавлены автосохранения и экстренные сохранения.

    Поддержка Discord Rich Precence.

    Добавлено Splash окно.

    Добавлен Content Browser.

    Цвет shape теперь снова зеленый.

    Исправлены группы обьектов.

    Добавлен Drag & Drop для level editor.

### Установка

Ставить поверх оригинального XRAY SDK 0.4 с заменой.

### Сборка

Проект настроен на сборку в Visual Studio 2019. Просто откройте .sln файл и выберите конфигурацию Release | x64.
